# Estilos (CSS)

Folhas de estilo da aplicação Liberty.

| Ficheiro | Uso |
|----------|-----|
| **main.css** | Estilos gerais (layout, tipografia, variáveis). |
| **components.css** | Componentes reutilizáveis (botões, cards, modais, inputs). |
| **liberty.css** | Estilos específicos da identidade Liberty. |
| **themes.css** | Temas (claro/escuro, cores). |
| **mobile.css** | Ajustes para ecrãs pequenos. |
| **responsive.css** | Media queries e layout responsivo. |

A ordem de carregamento no `index.html` define a precedência; normalmente `main.css` primeiro, depois `components.css` e os restantes.
